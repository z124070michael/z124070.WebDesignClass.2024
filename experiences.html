~ ~ ~ Project 1 - Chat Bot ~ ~ ~

I am building a chatbot together with other students from the Shibaura Institute of Technology, which will use artificial intelligence to assist international students. The student interacts with the bot and asks a question; the bot understands the question and provides the necessary information, making it much easier for international students to interface with an always-active operator capable of speaking any language. The bot will have GPT-like code and will only respond to university-related questions.

~ ~ ~ Project 2 - Web Monitoring ~ ~ ~

I am working on a project at my university in Italy together with a professor and an engineer, where we are trying to create a code for internal and external security for any website. By using tools like osquery and Velociraptor, we monitor all clients connected to a specific website and, if necessary, we can gather information and control the connected clients using these programs.
